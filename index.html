FalconMC

FalconMC is an afk Minecraft server written in Rust that aims to support all client versions ranging from 1.13 till 1.17.1 (1.8-12.2 coming soon). Its main priority is performance and the ability to effortlessly hold many players at once.
Writing custom plugins is supported.
